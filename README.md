# sanctuary-godot
🪽 A project to port Sanctuary from Unity to Godot Engine

Sanctuary is a project of a digital 3D notebook for personal journaling ([better solutions for scientific research exist](https://obsidian.md/)) following the idea of memory palaces and the spatial aids in both memorization and strenghtening emotional connection to important events in your life and things you want to keep close to your heart in moments of darkness.

![image](./docs/readme_media/sanctuary_header/sanctuary_header.png)

- Currently available [here on itch.io](https://eternalgarden.itch.io/sanctuary) for Windows, Mac (and Linux but I must clean up the project and finally push the functional Linux build).

## Why switching to Godot?

Sanctuary is my passion project that started already around 2 015, but my knowledge of software architecture was pretty limited then and the project collapsed twice untill the third version which achieved its current proof-of-concept stage. It is available for download through link above. The current Unity-based version has a very stable architecture that I am happy with, but I no longer trust in the future Unity licensing decisions and also firmly believe in the superiority of an open-sourced software, development of which is not possible in Unity, because it is itself a black box. This is especially important for a notemaking software where people put their private thoughts.
